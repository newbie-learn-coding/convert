Convert To It Free API (v1)

A simple, public, read-only API for developers. No API key required. Use it to discover supported formats and conversion handlers.

  • Public + no key
  • CORS enabled
  • Rate-limited (60 req/min/IP)

Endpoints

GET /api/v1/status

Service version & environment info.

curl -s https://converttoit.com/api/v1/status | jq

GET /api/v1/formats

List supported formats (aggregated across handlers).

Query params: category, from, to, q

curl -s "https://converttoit.com/api/v1/formats?category=image&from=true&q=png" | jq

GET /api/v1/handlers

List handlers and their supported formats.

curl -s https://converttoit.com/api/v1/handlers | jq
