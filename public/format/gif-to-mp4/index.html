FAQ

How much smaller is MP4 compared to GIF?

MP4 is typically 5-10 times smaller than GIF for the same animation, sometimes more.

Will my MP4 loop like a GIF?

Yes, but you need to configure loop settings. Some platforms handle looping differently for video.
